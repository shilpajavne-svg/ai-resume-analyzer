# AI Resume Parser & Analyzer

An automated career-tech utility built in Python that leverages the Google Gemini API to parse resume text data, analyze it against a specific target job role, and generate structural improvement insights.

## Features
- **ATS Compatibility Check:** Benchmarks your resume text against modern industry standards.
- **Score Generation:** Emits a direct score out of 100 based on keyword density and presentation.
- **Tailored Guidance:** Extracts specific recommendations based on your target role (e.g., Web Developer vs Data Scientist).

## Quick Execution
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Insert your Google API key inside `resume_analyzer.py`.
3. Launch the terminal tool:
   ```bash
   python resume_analyzer.py
   ```
